
# Company Research Assistant

This is a conversational AI agent that:
- Holds natural conversations
- Researches companies (placeholder offline version)
- Generates account plans
- Updates sections of plans
- Works in normal chat mode

## Setup
1. Install dependencies:
```
pip install -r requirements.txt
```

2. Add your API keys inside `.env` (optional for now).

3. Run the project:
```
python main.py
```
